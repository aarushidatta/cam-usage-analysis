# CAM Usage Analysis Across Age Groups

## Objective
To analyze patterns and determinants of CAM usage across different age groups using statistical and machine learning techniques.

## Methods
- Exploratory Data Analysis (EDA)
- Logistic Regression
- SMOTE for class balancing
- Chi-Square Test of Independence
- Hypothesis Testing
- Measures of Association (Yule’s Coefficient)

## Dataset
Survey-based demographic dataset containing healthcare usage and age-related variables.

## Tools
R, Python
